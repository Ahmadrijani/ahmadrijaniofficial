Setelan Tampilan
Tema background ada 4. Saat pilih Custom, warna tema bisa diatur dan semua tombol akan ikut menyesuaikan warna latar, tapi gradasinya tetap aktif.
Tema Background
Default
Tema asli, tidak diubah.
Abu
Warna latar abu gelap.
Hitam
Latar hitam pekat.
Custom
Warna tema bebas diatur.
Warna Aksen
Warna aksen mengatur highlight tombol, garis aktif, dan elemen penting. Default-nya biru, tapi bisa diganti.
Navigasi Mobile
Pilih navigasi bawaan untuk layar kecil. Default-nya navigasi bawah.
Navigasi Bawah
Tombol ada di bawah layar.
Hamburger
Menu disembunyikan di tombol garis tiga.
Muat Tab Saat Dibuka
Default aktif. Kalau dinyalakan, isi tab yang berat baru dimuat saat tab itu dibuka.
Muat tab saat dibuka
Tab lain tidak langsung load semua saat halaman dibuka.
Tutup iframe Saat Pindah Tab
Default mati. Kalau dinyalakan, iframe/tab yang tidak sedang dibuka akan ditutup supaya lebih enteng.
Tutup iframe jika tidak di tab itu
Iframe di tab lain akan dilepas saat kamu pindah tab.
Tampilan Hamburger

List

Grid
Tombol dibuat kotak 2 kolom.
Warna Tema Custom
Warna tombol akan otomatis mengikuti warna tema yang dipilih.
